SYSTEM REQUIREMENTS
═══════════════════════════════════════════════════════════════

Hardware:
  - Raspberry Pi 4B (4GB RAM)
  - USB Camera (Logitech)
  - LILYGO LoRa32 (ESP32 + SX1276)
  - 5V 3A Power Supply

Software:
  - Python 3.7+
  - Raspberry Pi OS (Bullseye/Bookworm)
  - torch, opencv-python, pyserial

Storage:
  - 16GB+ microSD card
  - ~500 MB RAM during operation
  - ~1 GB disk for models
